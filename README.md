🛍️ Kumar Brothers E-commerce Website

Kumar Brothers is a simple e-commerce web application developed to showcase and sell clothing products online. The website allows users to browse products, search for items, add products to the cart, and manage their shopping experience through an interactive interface.

This project is built using HTML, CSS, JavaScript, PHP, and MySQL with Apache (XAMPP) as the local server environment.

🚀 Features

User login and signup system

Product browsing for different categories (Men & Women)

Add to cart functionality

Product search feature

Responsive and user-friendly interface

Discount coupon support

Backend database to store user and product information

🛠️ Technologies Used
Frontend

HTML – Structure of the website

CSS – Styling and layout

JavaScript – Interactive functionality

Backend

PHP – Server-side scripting

Database

MySQL – Stores user and product data

Server Environment

Apache Server (XAMPP) – Local development server

⚙️ Setup Instructions

Install XAMPP.

Start Apache and MySQL from the XAMPP control panel.

Copy the project folder into:

xampp/htdocs/

Open phpMyAdmin.

Create a new database:

kumarbrothers

Import the provided .sql database file.

Open your browser and run:

http://localhost/KumarBrothers
